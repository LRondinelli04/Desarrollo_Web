    ANCHOS: PROPIEDAD WIDTH

    Es unicamente aplicable para elementos de bloque (ejemplo: imagenes, videos, etc). Determina el ANCHO de los elementos. Los valores posibles son: vh, vw, %, px, etc. Los valores que utlizaremos son: % y PX. EJEMPLO: width: 200px;

    ALTOS: PROPIEDAD HEIGHT

    Es unicamente aplicable para elementos de bloque (ejemplo: imagenes, videos, etc). Determina el ALTO de los elementos Los valores posibles son: vh, vw, %, px, etc. Los valores que utlizaremos son: % y PX Si no le indico un valor, este tomara el valor AUTO (predeterminado), lo que fluira con el contenido del elemento EJEMPLO: height: 200px; EJEMPLOS CON IMAGEN:

    MARGENES EN MI DOCUMENTO

    La propiedad MARGIN fija un espacio entre los elementos contiguos. Sus medidas pueden ser por porcentajes (%) o por medidas de longitud (PX). En esta se puede trabajar con valores negativos, aunque no sea lo mas comun. En caso de querer trabajar con valores negativos el trabajo que haria ese valor es ir acercando el elemento hacia el lado que estoy controlando en vez de alejarlo. EJEMPLO:

  • margin-left: 10px;
  • margin-right: 10px;
  • margin-bottom: 10px
  • margin-top: 15px;
