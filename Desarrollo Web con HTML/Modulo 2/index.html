ANCHOS: PROPIEDAD WIDTH

Es unicamente aplicable para elementos de bloque (ejemplo: imagenes, videos, etc). Determina el ANCHO de los elementos. Los valores posibles son: vh, vw, %, px, etc. Los valores que utlizaremos son: % y PX. EJEMPLO: width: 200px;

MARGENES EN MI DOCUMENTO

La propiedad MARGIN fija un espacio entre los elementos contiguos. Sus medidas pueden ser por porcentajes (%) o por medidas de longitud (PX). En esta se puede trabajar con valores negativos, aunque no sea lo mas comun. En caso de querer trabajar con valores negativos el trabajo que haria ese valor es ir acercando el elemento hacia el lado que estoy controlando en vez de alejarlo. EJEMPLO:

  • margin-left: 10px;
  • margin-right: 10px;
  • margin-bottom: 10px
  • margin-top: 15px;

Volver al principio

PADDING EN MI DOCUMENTO

Esta propiedad indica el espacio entre los bordes de un elemento y su contenido. A diferencia del MARGIN, no todos los elementos tienen PADDING. Se le puede asignar al igual que el MARGIN valores negativos

EJEMPLOS:
  • padding-top: 10px;
  • padding-left: -10px;
  • padding-right: -10px;
  • padding-bottom: 10px;
Volver al principio

BOX-SIZING

Permite o no que el PADDIGN se sume al ancho o alto total de un elemento. Por ejemplo, si a un elemento le agregamos: width 800px + padding 10px = el ancho total quedara en 820px. Es decir, 10px de PADDIGNde cada lado, en total 20px, se sumaran a 800px de width y el elemento sera mas grande que su width inicial.

Lo ideal en estos casos es trabajar con Box-Sizing y especificamente con su valor "border-box", para asi hacer que ese padding se tome el width inicial, lo que quedaria como: width 800px + padding 10px = quedara en 800px.

Esta propiedad la aplicaremos siempre al inicio de modificar un archivo con CSS, lo haremos con el selector universal * y aplicaremos la siguiente linea de codigo: box-sizing: border-box;

FONT-WEIGHT

Si queremos poner en Negrita un contenido lo podemos hacer con el font-weight. Si bien cuando utilizamo el Google Fonts ya podemos modificar y especificar que tipo de font-weight utilizar, para esto vamos a tener que importar todos aquellos que querramos. Sin embargo, desde CSS podemos aplicar font-weight: bold; para aplicar Negrita, y en caso contrario utilizamos font-weight: normal; para quitarlo.

En caso de haberlo importado de Google Fonts podriamos utilizar los numeros normalmente y evitar las palabras "bold" y "normal"

APLICANDO LA PROPIEDAD FONT-STYLE

Si queremos poner el contenido en Italica lo podemos hacer a traves de Font-Style, permitiendo poner el contenido en Italica o en normal como anteriormente el font-weight

Para esto utilizarmeos el font-style: italic; para aplicar el Italica, o el font-style: normal; para dejarlo con la tipografia normal

Imagenes

Las imagenes son de los elementos mas importantes de la Web. Para poder llamar a la etiqueta de las imagenes se escribe la siguiente linea de codigo: IMG. Dentro de este atributo se ven 2 opciones, el SRC, el cual es similar a un HREF ya que se indica la direccion donde se encuentra la imagen (esta puede ser el link de una pagina o la direccion de una carpeta), y ALT la cual permite describir el contenido de la imagen (por ejemplo cuando no carga una imgaen y se pasa el mouse por arriba de esta aparece un texto, el cual es el ALT).

Los formatos permitidos son: PNG, GIF, JPG y SVG. Los que mejor calidad tienen son PNG y SVG.

A las imagenes tambien se las puede transformar en un vinculo. Ademas de aplicarle width para el ancho y height para la altura.
