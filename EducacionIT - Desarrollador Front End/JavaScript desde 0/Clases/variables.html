Variables en JavaScript

Para llamar una variable en JavaScript se usa el termino "let (nombre de variable)" y a este se le puede asignar un valor con = (valor a asignar) o se puede dejar sin asignar un especifico.

Condicionales en JavaScript

Para saber si en un prompt se ingreso una palabra pero estamos en duda si el usuario utilizo mayusculas o minusculas, lo que se debe hacer es transformar este texto en mayusculas o minusculas con el siguiente codigo:

let mensaje = prompt('texto');
mensaje.toLowerCase > lo pasa a minuscula
mensaje.toUpperCase > lo pasa a mayuscula

Esto lo transforma a todo el texto y luego para poder comparar es mas simple ya que solo necesitaremos una condicion y no preguntar por todos los caracteres.

Ubicar Selectores en JavaScript

Para ubicar de manera correcta un elemento en JavaScript nosotros lo veniamos haciendo con el siguiente codigo: "nombreID.innerText = ''".
Sin embargo, esto no es lo mas eficiente ya que en el documentonos podemos encontrar tanto como con IDs o Class, para esto utilizaremosla siguiente codigo:
"const mensaje = document.querySelector('#mensaje') > en caso de que sea un ID"
"const mensaje = document.querySelector('.mensaje') > en caso de que sea una Clase"
