Tipos de datos

En TypeScript se pueden declarar variables con los siguientes tipos de datos:

  • Primitivos
    • String
    • Number
    • Boolean
    • Null
    • Undefined
  • Compuestos
    • Object
    • Array
    • Enum
    • Function
  • Definidos por el usuario
    • Class
    • Interface
    • Type
